                //Git
add = prepara cambios
commit = guardar version
push = subir a la nube
pull = traer cambios

1)
git add .
git commit -m "Describe lo que hiciste"
git push

🔹 1. Deshacer el último push (manteniendo historial limpio)

Si ya hiciste push y quieres revertir los cambios sin borrar el historial:

git revert HEAD
git push

👉 Crea un commit nuevo que deshace el anterior.
✔ Seguro si ya trabajas con más gente.

🔹 2. Borrar el último commit del repositorio remoto (reescribir historia)

Si quieres literalmente “como si nunca hubiera pasado”:

git reset --hard HEAD~1
git push --force

👉 Elimina el último commit local y lo sobrescribe en remoto.
⚠️ Peligroso si alguien más ya hizo pull.

🔹 3. Deshacer varios commits
git reset --hard HEAD~3
git push --force

👉 Borra los últimos 3 commits.


                        //FORMATEO DE TEXTO EN PHP
%s y %d son especificadores de formato utilizados en funciones como sprintf() y printf()
en PHP para insertar valores en cadenas de texto de forma segura y controlada. 

%s indica que se debe insertar un valor de tipo cadena (string) en la posición donde aparece.  
Por ejemplo, sprintf("Hola %s", "Ana") devuelve "Hola Ana".

%d indica que se debe insertar un valor entero (integer) en la posición correspondiente.  
Por ejemplo, sprintf("Tengo %d años", 25) devuelve "Tengo 25 años". 

El especificador %.2f en PHP se utiliza para formatear un número de punto flotante con
exactamente dos decimales.

//PHP Operador de cohalecencia nula "??" 
El?? El operador se llama operador de coalescencia nula. Devuelve el operando izquierdo si el operando no es nulo;
de lo contrario devuelve el operando de la derecha.

$rabbits;
$rabbits = count($somearray);

if ($rabbits == null) {
    $rabbits = 0;
}


$rabbits = $rabbits ?? 0;


Operadores modernos más usados
?? (null coalescing), ?: (Elvis), y ?-> (nullsafe) 
son operadores que usarás constantemente en código PHP moderno. Hacen el código más limpio y seguro.
